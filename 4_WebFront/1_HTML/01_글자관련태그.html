글자 관련 기타 태그

b 태그 : 단순히 글자를 굵게 표시하는 태그
strong 태그 : 글자를 강조하는 태그(차이점 : 웹 접근성)
i 태그 : 단순히 글자를 기울이는 태그
em 태그 : (웹 접근성)글자를 기울이는 태그
mark 태그 : 형광펜 효과를 나타내는 태그
u 태그 : 밑줄 긋는 태그
s 태그 : 취소선을 긋는 태그
del 태그 : (웹 접근성) 스크린 리더가 읽지 않는다.
small 태그 : 글자를 작게 표현하는 태그
sup 태그(윗첨자) : x2
sub 태그(아래첨자) : log10
abbr(abbreviation : 약어) 태그 : 마우스 오버시 툴팁 출력 태그 HTTP

글자 관련 태그 응용

태그들은 해당 태그 내에서 중첩으로 사용 가능하다.
굵은 글자를 사용할 수도 있고, 기울이거나,취소선을 넣는 등
다양하게 사용할 수 있다.
